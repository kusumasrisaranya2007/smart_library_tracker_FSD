# Smart Library Tracker

A complete, smart library management system built fully on the frontend using Vanilla HTML, CSS, and JavaScript. The application leverages **LocalStorage** for data persistence and features a **Retro-Futurist Glitch Art** interface.

## 🚀 Features

- **Splash Screen & Animated Logo:** Immersive entry point with glitch animations.
- **Premium Dashboard:** Real-time stats (Total Books, Available, Borrowed, Overdue) and analytical charts using Chart.js.
- **Glassmorphism UI:** Modern, translucent UI components with jarring color contrasts.
- **Full Library CRUD:** Create, Read, Update, and Delete books, borrowers, and categories.
- **Borrow/Return System:** Automated overdue detection and fine calculation.
- **Reports & Analytics:** Visual insights into library activity.
- **Smart Features:**
  - Dark/Light mode toggle.
  - Keyboard shortcuts.
  - Search suggestions and filtering.
  - Export data to CSV.
  - Print-ready reports.

## 🛠️ Technologies Used

- **HTML5:** Core structure.
- **CSS3:** Advanced glitch animations, glassmorphism, and responsive layout.
- **JavaScript (ES6+):** Application logic, DOM manipulation, and LocalStorage integration.
- **Chart.js:** Data visualization.
- **Font Awesome:** Iconography.

## 📂 Folder Structure

```
/
```

## ⚙️ How to Run

1. Open `index.html` in any modern web browser.
2. The application is entirely client-side; no server is required for development.
